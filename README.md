# AssignTrack - Student assignment tracker (TASK MANAGER)

# PROJECT DESCRIPTION 
AssignTrack is a web-based system designed to help students manage assignments, subjects, deadlines, and track academic progress efficiently. The system include an interactive dashboard, statical charts, assignment management, calendar view , and dark/light theme support. 

# Main features 
Authentication System
Login & Register
Password hashing using PHP
Session management
Error validation

# Dashboard
Assignment statistics
Deadline countdown
High priority tasks
Subject progress tracking
Interactive charts using Chart.js

# Assignments Page
Add, edit, and delete assignments
Search functionality
Pending/completed status
Priority levels (High, Medium, Low) 

# Subjects Page
Subject management
Subject progress tracking
Assignment statistics by subject 

# Calendar Page
Interactive calendar view
Assignment indicators
Upcoming deadline reminders

# Statistics Page
Assignment progress charts
Priority analysis
Subject performance comparison

 # Settings Page
Dark/light mode
Export & import data
PDF report generation
Logout system

# Support Page
FAQ section
Privacy policy
Terms of use
Contact information
